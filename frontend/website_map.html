Step 2

The website flow is organized into the following pages and actions:

1) User Details

Collect Gmail and phone number and create a unique session ID for temporary conversation tracking.

2) Website Map

Explains the full page flow and core functionality before entering the chat workspace.

3) Chat Workspace

Upload PDF, ask real-time questions, add annotations, edit sent messages, and download reports.

4) Temporary Memory

Conversation data is stored in MongoDB while chat is active and deleted when the chat session ends.
